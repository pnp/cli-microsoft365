# spo user list

Lists all the users within specific web

## Usage

```sh
m365 spo user list [options]
```

## Options

```md definition-list
`-u, --webUrl <webUrl>`
: URL of the web to list the users from
```

<Global />

## Examples

Get list of users in a web

```sh
m365 spo user list --webUrl https://contoso.sharepoint.com/sites/project-x
```
