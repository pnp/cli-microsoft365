# aad user recyclebinitem list

Lists users from the recycle bin in the current tenant

## Usage

```sh
m365 aad user recyclebinitem list [options]
```

## Options

--8<-- "docs/cmd/_global.md"

## Examples

List all removed users

```sh
m365 aad user recyclebinitem list
```
