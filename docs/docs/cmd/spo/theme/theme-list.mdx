# spo theme list

Retrieves the list of custom themes

## Usage

```sh
m365 spo theme list [options]
```

## Options

<Global />

## Remarks

:::info

To use this command you must be either **SharePoint Administrator** or **Global Administrator**.

:::

## Examples

List available themes

```sh
m365 spo theme list
```
