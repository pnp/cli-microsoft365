# spo tenant commandset list

Get a list of ListView Command Sets that are installed tenant-wide

## Usage

```sh
spo tenant commandset list [options]
```

## Options

<Global />

## Examples

Retrieves a list of ListView Command Sets.

```sh
m365 spo tenant commandset list
```
