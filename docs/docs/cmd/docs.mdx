# docs

Returns the CLI for Microsoft 365 docs webpage URL

## Usage

```sh
m365 docs [options]
```

## Options

<Global />

## Remarks

Configure `autoOpenLinksInBrowser` using [cli config set](../cmd/cli/config/config-set.mdx) to automatically open the webpage in the default browser.

## Examples

Returns the CLI for Microsoft 365 docs webpage URL

```sh
m365 docs
```
