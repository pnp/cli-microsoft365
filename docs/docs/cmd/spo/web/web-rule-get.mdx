# spo web rule get

Retrieves details of a specific rule from a SharePoint site list

## Usage

```sh
m365 spo web rule get [options]
```

## Options

```md definition-list
`-u, --webUrl <webUrl>`
: The URL of the SharePoint site.

`--id <id>`
: The ID of the rule.
```

<Global />

## Examples

Retrieve a rule by ID

```sh
m365 spo web rule get --webUrl https://contoso.sharepoint.com/sites/Marketing --id 7cbb4c8d-8e4d-4d2e-9c6f-3f1d8b2e6a0e
```
