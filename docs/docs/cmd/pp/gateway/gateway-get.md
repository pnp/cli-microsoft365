# pp gateway get

Get information about the specified gateway

## Usage

```sh
m365 pp gateway get [options]
```

## Options

`-i, --id <id>`
: ID of the Gateway.

## Examples

Get information about the specified gateway.

```sh
m365 pp gateway get --id 1f69e798-5852-4fdd-ab01-33bb14b6e934
```
