# todo list list

Returns a list of Microsoft To Do task lists

## Usage

```sh
m365 todo list list [options]
```

## Options

--8<-- "docs/cmd/_global.md"

## Examples

Get the list of Microsoft To Do task lists

```sh
m365 todo list list
```
