# spp contentcenter list

Gets information about the SharePoint Premium content centers

## Usage

```sh
spp contentcenter list
```

## Options

No options required

<Global />

## Remarks

:::info

To use this command you have to have permissions to access the tenant admin site.

:::

## Examples

Gets information about the SharePoint Premium content centers

```sh
m365 spp contentcenter list
```
