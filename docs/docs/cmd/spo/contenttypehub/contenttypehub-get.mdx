# spo contenttypehub get

Returns the URL of the SharePoint Content Type Hub of the Tenant

## Usage

```sh
m365 spo contenttypehub get [options]
```

## Options

<Global />

## Examples
  
Retrieve the Content Type Hub URL

```sh
m365 spo contenttypehub get
```
