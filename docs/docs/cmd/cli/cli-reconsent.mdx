# cli reconsent

Returns Azure AD URL to open in the browser to re-consent CLI for Microsoft 365 permissions

## Usage

```sh
m365 cli reconsent [options]
```

## Options

<Global />

## Examples

Get the URL to open in the browser to re-consent CLI for Microsoft 365 permissions

```sh
m365 cli reconsent
```
