# entra multitenant get

Gets properties of the multitenant organization

## Usage

```sh
m365 entra multitenant get [options]
```

## Options

<Global />

## Remarks

:::info

To use this command you must be at least **Security Administrator**.

:::

## Examples

Gets properties of the multitenant organization

```sh
m365 entra multitenant get
```
