# planner task get

Retrieve the details of the specified planner task

## Usage

```sh
m365 planner task get [options]
```

## Options

`--id <id>`
: ID of the task to retrieve details from

## Examples

Retrieve the details of the specified planner task

```sh
m365 planner task get --taskId 'vzCcZoOv-U27PwydxHB8opcADJo-'
```
