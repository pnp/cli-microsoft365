# spo theme list

Retrieves the list of custom themes

## Usage

```sh
m365 spo theme list [options]
```

## Options

<Global />

## Remarks

:::info
To use this command you have to have permissions to access the tenant admin site.
:::

## Examples

List available themes

```sh
m365 spo theme list
```
