# spo homesite set

Configures the specified SharePoint Home Site setting options such as draft mode, audiences, targeted license type, and Viva Connections default start.

## Usage

```sh
m365 spo homesite set [options]
```

## Options

```md definition-list
`-u, --siteUrl <siteUrl>`
: The URL of the Home Site

`--draftMode [draftMode]` 
: Set draft status. Valid values are: `true`, `false`.

`--audienceIds [audienceIds]` 
: Comma-separated list of M365 group Ids to set as audiences. Specify either `audienceIds` or `audienceNames`, but not both.

`--audienceNames [audienceNames]` 
: Comma-separated list of M365 group names or Active Directory security groups to set as audiences. Specify either `audienceIds` or `audienceNames`, but not both.

`--targetedLicenseType [targetedLicenseType]` 
: Value of either `everyone`, `frontLineWorkers` or `informationWorkers`.

`--vivaConnectionsDefaultStart [vivaConnectionsDefaultStart]`
: Specifies whether the home site is the default start for Viva Connections. Accepts `true` or `false`.
```

<Global />

## Remarks

:::info

To use this command you must be a Global or SharePoint administrator.

:::

## Examples

Sets the Home site to the provided site collection url and sets the Viva Connections landing experience to the SharePoint home site

```sh
m365 spo homesite set --siteUrl https://contoso.sharepoint.com/sites/comms --vivaConnectionsDefaultStart true
```

Set the Home site in draft mode

```sh
m365 spo homesite set -siteUrl https://contoso.sharepoint.com/sites/contosoportal --draftMode true
```

Set the audience to everyone

```sh
m365 spo homesite set -siteUrl https://contoso.sharepoint.com/sites/contosoportal --targetedLicenseType "everyone"
```

Set the audience to M365 groups using M365 Groups Ids

```sh
m365 spo homesite set -u https://contoso.sharepoint.com/sites/contosoportal --audienceIds "978b5280-4f80-47ea-a1db-b0d1d2fb1ba4,21af775d-17b3-4637-94a4-2ba8625277cb"
```

Set the audience to M365 groups or Azure Directory security groups using names 

```sh
m365 spo homesite set -u https://contoso.sharepoint.com/sites/contosoportal --audienceNames "Marketing, IT department"
```

Clear the audience and set to draft 

```sh
m365 spo homesite set -u https://contoso.sharepoint.com/sites/contosoportal --audienceNames '' --draftMode true
```


Configure multiple options

```sh
m365 spo homesite set -u https://contoso.sharepoint.com/sites/contosoportal --audienceNames 'Marketing, IT department' --draftMode true --targetedLicenseType "everyone" --vivaConnectionsDefaultStart true
```
