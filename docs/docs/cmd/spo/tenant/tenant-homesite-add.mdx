# spo tenant homesite add

Add a Home Site.

## Usage

```sh
m365 spo tenant homesite add [options]
```

## Options

<Global />

## Examples

Add a Home Site

```sh
m365 spo tenant homesite add --url "https://contoso.sharepoint.com/sites/testcomms"

```
