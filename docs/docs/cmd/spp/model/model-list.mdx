# m365 spp model list

Retrieve the list of SharePoint Premium unstructured document processing models on the content center site

## Usage

```sh
m365 spp model list [options]
```

## Options

```md definition-list
`-u, --siteUrl <siteUrl>`
: The URL of the content center site.
```

<Global />

## Remarks

:::note
You can only create one Trial Container per tenant, a single application registration can only contain one Container and a tenant can contain a maximum of five Containers in total.
:::

## Examples

Retrieve the list of SharePoint Premium unstructured document processing models on the content center site.

```sh
m365 spp model list --siteUrl https://contoso.sharepoint.com/sites/ContentCenter
```
