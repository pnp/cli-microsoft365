# entra m365group user set

Updates role of the specified user in the specified Microsoft 365 Group or Microsoft Teams team

## Usage

```sh
m365 entra m365group user set [options]
```

## Alias

```sh
m365 aad m365group user set [options]
```

```sh
m365 teams user set [options]
```

## Options

```md definition-list
`-i, --groupId [groupId]`
: The ID of the Microsoft 365 group for which to update the user. Specify either `groupId`, `groupDisplayName` or `teamId`, but not multiple.

`--groupDisplayName [groupDisplayName]`
: Display name of the Microsoft 365 Group for which to update the user. Specify either `groupId`, `groupDisplayName` or `teamId`, but not multiple.

`--teamId [teamId]`
: The ID of the Microsoft Teams team for which to update the user. Specify either `groupId`, `groupDisplayName` or `teamId`, but not multiple.

`-n, --userName <userName>`
: UPN of the user for whom to update the role (eg. johndoe@example.com)

`-r, --role <role>`
: Role to set for the given user in the specified Microsoft 365 Group or Microsoft Teams team. Allowed values: `Owner`, `Member`.
```

<Global />

## Remarks

The command will return an error if the user already has the specified role in the given Microsoft 365 Group or Microsoft Teams team.

## Examples

Promote the specified user to owner of the given Microsoft 365 Group

```sh
m365 entra m365group user set --groupId '00000000-0000-0000-0000-000000000000' --userName 'anne.matthews@contoso.onmicrosoft.com' --role Owner
```

Demote the specified user from owner to member in the given Microsoft 365 Group

```sh
m365 entra m365group user set --groupDisplayName Finance --userName 'anne.matthews@contoso.onmicrosoft.com' --role Member
```

Promote the specified user to owner of the given Microsoft Teams team

```sh
m365 teams user set --teamId '00000000-0000-0000-0000-000000000000' --userName 'anne.matthews@contoso.onmicrosoft.com' --role Owner
```

Demote the specified user from owner to member in the given Microsoft Teams team

```sh
m365 teams user set --teamId '00000000-0000-0000-0000-000000000000' --userName 'anne.matthews@contoso.onmicrosoft.com' --role Member
```

## Response

The command won't return a response on success.
